# CV Setup Instructions

## How to Add Your CV

1. Place your CV PDF file in the root directory of this website
2. Name it exactly: `cv.pdf`
3. The CV link in the navigation bar will automatically work
4. When visitors click "CV", the PDF will open in a new browser tab

## File Location

```
.
└── cv.pdf  ← Place your CV file here
```

## Note

Make sure the file is named `cv.pdf` (lowercase) to match the link in the navigation.
